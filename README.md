## Cadres analytiques

Les scénarios croisent les travaux de :

- **Jean-Marc Jancovici** — descente énergétique, EROI
- **Philippe Bihouix** — mur des métaux, low-tech
- **Olivier Hamant** — robustesse, variabilité
- **Arthur Keller** — vulnérabilité systémique, résilience par déconnexion, travail des récits
- **GIEC / CMIP7** — forçages climatiques, trajectoires d'émissions
